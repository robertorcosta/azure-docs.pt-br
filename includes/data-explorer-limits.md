A tabela a seguir descreverá os limites máximos para os clusters do Azure Data Explorer.

| Recurso | Limite |
| --- | --- |
| Clusters por região e por assinatura | 20 |
| Instâncias por cluster | 1000 | 
| Número de bancos de dados em um cluster | 10.000 |
| Número de configurações de bancos de dados anexadas em um cluster | 70 |

A tabela a seguir descreverá os limites das operações de gerenciamento executadas em clusters do Azure Data Explorer.

| Escopo | Operação | Limite |
| --- | --- | --- |
| Cluster | ler (por exemplo, obter um cluster) | 500 a cada 5 minutos |
| Cluster | gravar (por exemplo, criar um banco de dados) | 1\.000 por hora |
