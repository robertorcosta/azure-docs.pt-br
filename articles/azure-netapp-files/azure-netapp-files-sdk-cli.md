## <a name="azure-resource-manager-templates"></a>Modelos do Azure Resource Manager  

Azure Resource Manager (ARM) permite provisionar seus aplicativos usando um modelo declarativo. Em um modelo único, você pode implantar vários serviços, juntamente com suas dependências. Use o mesmo modelo para implantar repetidamente seu aplicativo durante cada estágio do ciclo de vida do aplicativo.   

Consulte [modelos de ARM disponíveis para Azure NetApp files](https://azure.microsoft.com/resources/templates/?term=anf).
