# <a name="supported-language-and-runtime-versions-for-apache-spark-and-dependent-components"></a>Língua suportada e versões em tempo de execução para Apache Spark e componentes dependentes  

O serviço Apache Spark no Azure Synapse Analytics suporta vários tempos de execução diferentes e serviços este documento lista as versões.

## <a name="spark-version"></a>Versão spark

Spark 2.4 é a versão suportada atualmente.

## <a name="operating-system-version"></a>Versão do sistema operacional

Apache Spark in Azure Synapse é executado no Ubuntu versão 16.04.

## <a name="java-version"></a>Versão Java

Apache Spark in Azure Synapse inclui java versão 1.8.0_222

## <a name="scala-version"></a>Versão da escala

Apache Spark em Azure Synapse inclui Scala 2.11.12

## <a name="net-core-version"></a>Versão do .NET Core

Apache Spark em Azure Synapse inclui .NET Core 3.0

## <a name="net-support-for-apache-spark"></a>.NET Suporte para Apache Spark

Azure Synapse inclui suporte .NET para Apache Spark 0.5.0

## <a name="delta-lake"></a>Delta Lake

Synapse Analytics inclui suporte para Delta Lake 0.5.0

## <a name="python-version"></a>Versão do Python

Apache Spark in Azure Synapse inclui uma instalação do Python versão 3.6.1
