# <a name="jenkins-plugins-for-azure"></a>Plug-ins do Jenkins para o Azure

Os plug-ins Jenkins a seguir dão suporte a vários recursos para uso com o Azure.
  
| Plug-in Jenkins | DESCRIÇÃO                                   |
|------------------------------------------------------------------------------|
| [Plug-in de serviço Azure App](https://plugins.jenkins.io/azure-app-service)     | Plug-in Jenkins para implantar um serviço de Azure App (atualmente, dá suporte apenas ao aplicativo Web). | 
| [Plug-in do Azure AD](https://plugins.jenkins.io/azure-ad)                       | Plug-in Jenkins que dá suporte à autenticação & autorização via Azure Active Directory. | 
| [Plug-in Gerenciador de artefatos do Azure](https://plugins.jenkins.io/azure-artifact-manager) | O plug-in do Gerenciador de artefatos do Azure é um Gerenciador de artefato que permite armazenar seus artefatos no armazenamento de BLOBs do Azure. O plug-in do Gerenciador de artefatos do Azure funciona de forma transparente para Jenkins e seus trabalhos, é como o Gerenciador de artefatos padrão. | 
| [Plug-in de Agentes do Contêiner do Azure](https://plugins.jenkins.io/azure-container-agents) | O plug-in de agentes de contêiner do Azure pode ajudá-lo a executar um contêiner como um agente no Jenkins | 
| [Plug-in de tarefa do registro de contêiner do Azure](https://plugins.jenkins.io/azure-container-registry-tasks)       | Plug-in Jenkins para enviar uma solicitação de Build do Docker para o [registro de contêiner do Azure](/azure/container-registry/container-registry-tasks-overview). |
| [Plug-in do Serviço de Contêiner do Azure](https://plugins.jenkins.io/azure-acs)       | Plug-in Jenkins para implantar configurações no AKS (serviço de contêiner do Azure). | 
| [Plug-in de Credencial do Azure](https://plugins.jenkins.io/azure-credentials)      | Plug-in Jenkins para gerenciar as credenciais do Azure. | 
| [Plug-in da Função do Azure](https://plugins.jenkins.io/azure-function)           | Plug-in Jenkins para implantar uma função do Azure. | 
| [Plug-in Azure IoT Edge](https://plugins.jenkins.io/azure-iot-edge)           | Azure IoT Edge plug-in facilita a configuração de um pipeline de CI/CD para o desenvolvimento de uma solução de IoT Edge no Jenkins. | 
| [Plug-in do Service Fabric do Azure](https://plugins.jenkins.io/service-fabric)     | Plug-in Jenkins para projetos do Azure Service Fabric do Linux. |
| [Plug-in de Armazenamento do Azure](https://plugins.jenkins.io/windows-azure-storage)     | plug-in para carregar artefatos de compilação ou baixar dependências de compilação do Microsoft Azure armazenamento de BLOBs. | 
| [Plug-in de agentes de VM do Azure](https://plugins.jenkins.io/azure-vm-agents)         | Plug-in Jenkins para criar agentes Jenkins em máquinas virtuais do Azure (por meio do modelo de Azure Resource Manager). | 
| [Plug-in de conjunto de escala de máquina virtual do Azure](https://plugins.jenkins.io/azure-vmss)           | Plug-in Jenkins para implantar imagens de VM em conjuntos de dimensionamento de máquinas virtuais do Azure. |
