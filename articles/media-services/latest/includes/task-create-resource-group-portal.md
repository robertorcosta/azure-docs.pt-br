## <a name="create-a-resource-group"></a>Criar um grupo de recursos

1. Na tela Página Inicial do portal do Azure, selecione **Criar um recurso**. A tela do Marketplace será exibida.
1. Selecione **Grupos de recursos**.  Uma lista de grupos de recursos será exibida.
1. Selecione **Adicionar**. A tela Criar um grupo de recursos será exibida.
1. Selecione a assinatura que deseja usar para este grupo de recursos.
1. Insira o nome do grupo de recursos no campo **Grupo de recursos**.
1. Escolha uma **Região** para o grupo de recursos.
1. Selecione **Examinar + criar**.
