# <a name="azure-storage-data-governance-management-and-migration-partners"></a>Parceiros de governança, gerenciamento e migração de dados do armazenamento do Azure

Este artigo destaca as empresas parceiras da Microsoft integradas ao armazenamento do Azure que podem melhorar seus recursos gerais de gerenciamento de dados. Essas soluções de parceiros podem dar suporte à avaliação de armazenamento e relatório, migração independente de plataforma, replicação, camadas de nuvem ou governança de dados.
