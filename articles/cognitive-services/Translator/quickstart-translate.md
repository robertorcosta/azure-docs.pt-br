# <a name="quickstart-use-the-translator-to-translate-text"></a>Início Rápido: Usar o Tradutor para traduzir texto

Neste início rápido, você aprenderá a traduzir uma cadeia de texto de inglês em alemão, italiano, japonês e tailandês usando a API REST do Tradutor.

Este início rápido requer uma [conta dos Serviços Cognitivos do Azure](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account) com um recurso Tradutor.
