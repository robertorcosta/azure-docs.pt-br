# <a name="authoring-api-changes"></a>Criando alterações de API 

As APIs de criação são atualizadas, para v3, para permitir a [decomposição do modelo com entidades aprendidas por máquina](luis-concept-data-extraction.md). 

Você pode criar as APIS v2 ou Preview v3. O mesmo aplicativo também pode optar por usar respostas de ponto de extremidade de previsão v2 ou v3. Você não precisa ter a mesma versão da API para a criação e a resposta de ponto de extremidade de previsão.

O portal do LUIS usa exclusivamente as APIs de criação v3.

Examine [as alterações no ponto de extremidade de previsão para a](luis-migration-api-v3.md) API v3.
