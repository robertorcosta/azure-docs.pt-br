# <a name="workloads-on-azure-virtual-machines"></a>Cargas de trabalho em máquinas virtuais do Azure

As máquinas virtuais no Azure podem dar suporte a várias cargas de trabalho. Cada carga de trabalho pode ter orientações especializadas sobre o design ideal para implantações em VMs no Azure. Aqui estão algumas das diretrizes que podemos oferecer para cargas de trabalho especializadas.

- A [computação de alto desempenho](./hpc/overview.md) no Azure usa máquinas virtuais otimizadas de CPU ou GPU para resolver problemas complexos de computação intensa.
- O [SAP](./sap/get-started.md) no Azure pode executar suas cargas de trabalho e cenários SAP de missão crítica em uma plataforma escalonável, em conformidade e comprovada pela empresa.
- O [Oracle](./oracle/oracle-considerations.md) dá suporte à execução do Oracle dB 12,1 Standard e Enterprise Editions no Azure em imagens de máquina virtual com base em Oracle Linux.
- O [Red Hat](./redhat/redhat-overview.md) no Azure tem suporte por meio de uma variedade de ofertas de imagem Red Hat Enterprise Linux e da infraestrutura de atualização do Red Hat para fornecer atualizações sob demanda.
- [Hospedagem de mainframe](./mainframe-rehosting/overview.md)
