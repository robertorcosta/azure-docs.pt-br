# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>APIs geoespaciais para adicionar mapas, análises espaciais e soluções de mobilidade com mapas do Azure

O [mapas do Azure](https://azure.microsoft.com/services/azure-maps/) fornece a plataforma de local nativamente na nuvem do Azure, oferece um portfólio de serviços geoespaciais que usam dados de mapeamento atualizados para fornecer um contexto geográfico preciso para aplicativos Web e móveis. Ele permite que desenvolvedores, empresas e ISVs criem aplicativos com reconhecimento de local e soluções de IoT, mobilidade, logística e acompanhamento de ativos.

## <a name="azure-maps-features"></a>Recursos do Azure Maps
- A **pesquisa** fornece detalhes sobre POI (pontos de interesse próximos), renderizar locais em um mapa e endereçar geocódigo (ou geocódigo invertido) para obter coordenadas de latitude/longitude correspondentes.
- Melhore o tempo de viagem e evite engarrafamentos usando o **tráfego**.
- Integre **mapas** claros e fáceis de ler em sua solução.
- O **Roteamento** para apresentar as rotas mais curtas ou mais rápidas disponíveis — para vários destinos de cada vez — ou fornecer roteiros e fatos especializados para os ciclistass, os veículos e as propagandas comerciais.
- Os **fusos horários** tornam mais fácil para os usuários ver qual é o tempo em qualquer lugar do mundo.
- **Operações espaciais** para aprimorar sua inteligência de local.
- **Geolocalização** para pesquisar o país/região de um endereço IP.
- **Serviço de dados** para carregar e armazenar dados geoespaciais para uso com operações espaciais ou composição de imagem.
- A **mobilidade** fornece inteligência de local em tempo real sobre serviços de trânsito públicos próximos, incluindo interrupções, informações de rota e estimativas de tempo de viagem.
