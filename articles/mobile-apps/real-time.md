# <a name="build-real-time-applications-with-azure-signalr"></a>Crie aplicativos em tempo real com o Azure Signalr

O [signalr do Azure](https://azure.microsoft.com/services/signalr-service/) permite que você adicione facilmente funcionalidades em tempo real aos aplicativos. Essa funcionalidade em tempo real permite que o serviço envie por push atualizações de conteúdo para aplicativos móveis conectados. Como resultado, os aplicativos são atualizados sem a necessidade de sondar o servidor ou enviar novas solicitações HTTP para atualizações.

## <a name="azure-signalr-features"></a>Recursos do Azure Signalr
- **Amplo suporte a clientes** – ele funciona com uma ampla variedade de clientes, como navegadores Web e móveis, aplicativos de desktop, processo de servidor, dispositivos IOT e consoles de jogos.
- O serviço signalr permite que **várias instâncias trabalhem juntas** para dimensionar para milhões de conexões de aplicativos móveis. Ele também dá suporte a várias regiões globais para fins de fragmentação, alta disponibilidade ou recuperação de desastre.
- **Ofereça APIs avançadas para diferentes padrões de mensagens** para enviar mensagens para uma conexão específica, todas as conexões ou um subconjunto de conexões que pertencem a um usuário específico ou que foram colocadas em um grupo arbitrário.
